How To Use This Workspace

Celeste's operating guide. Read once. Reference as needed. This workspace runs on one rule: repeatable work becomes a machine.

The Core Operating Rule
01
Repeatable work gets automated.
If you do it more than twice, build a content engine for it.
02
Original work gets documented.
Custom work lives in Archive. Style notes and prompts get saved so it can be recreated.
03
Working content becomes an engine.
When something performs, document the process, template it, and build a repeatable system.
04
Automation gets reviewed.
AI does not publish by itself. Every output passes through Automation Review before distribution.
05
Distribution is part of completion.
A graphic is not done because it was made. Done = published, tracked, archived.
06
Metrics decide what we repeat.
Every piece gets measured. Winners become engines. Low performers get retired or revised.
07
Time blocks keep creative work moving.
No task gets unlimited time. When the block ends, you choose what happens next.
